# 🎲 Yacht Dice Game

콘솔 환경에서 동작하는 **Yacht Dice 게임**으로,  
C++을 활용해 점수 계산 로직과 콘솔 UI 출력 제어를 구현한 프로젝트입니다.

---

## 📌 프로젝트 개요 (Overview)

- 주사위를 굴려 점수를 획득하는 Yacht Dice 게임을 콘솔 기반으로 구현
- 게임 규칙에 따른 점수 계산과 사용자 입력 흐름을 중심으로 개발
- 콘솔 환경에서도 가독성과 사용자 경험을 고려한 UI 출력 방식 적용

---

## 👥 프로젝트 정보

- **프로젝트 유형** : 개인 프로젝트
- **개발 환경**
  - **Language** : C++
  - **IDE** : Visual Studio
  - **Platform** : Console (Windows)

---

## 🛠️ 주요 기능 (Features)

- 주사위 굴림 및 재굴림 기능
- Yacht Dice 규칙에 따른 점수 계산 로직 구현
- 점수판, 주사위, 메시지 영역을 분리한 콘솔 UI 출력
- 게임 진행 상태에 따른 사용자 입력 처리
- 라운드 종료 및 최종 점수 계산 기능

---

## 👩‍💻 담당 역할 (My Role)

- **C++ 기반 콘솔 Yacht Dice 게임 로직 설계 및 구현**
  - 주사위 굴림, 재굴림, 점수 확정 등 게임 핵심 로직 구현
  - 족보 판정 및 점수 계산 알고리즘 설계

- **콘솔 좌표 기반 UI 출력 구조 설계**
  - 콘솔 좌표를 활용하여 점수판, 주사위, 메시지 영역을 분리한 UI 레이아웃 구성
  - 전체 화면 초기화 없이 필요한 영역만 갱신하도록 출력 구조 구현

- **게임 진행 흐름 제어 및 입력 처리**
  - 키 입력에 따라 주사위 고정, 재굴림, 점수 확정 처리
  - 라운드 진행 상태, 남은 굴림 횟수, 족보 사용 여부에 따라 화면 동적 갱신
